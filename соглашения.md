
# Соглашения по шрифтам

* Команды монитора-отладчика выделяются **жирным шрифтом**, все команды даны в нижнем регистре (пример: **menable**)
* Блоки микроконтроллера, наименования изделий выделяются **жирным шрифтом** (пример: **HET**)
* Ссылки на другие ресурсы и названия сторонних программ выделяются **жирным шрифтом** (пример: [**DRV8301-HC-EVM rev.D**](http://www.ti.com/tool/DRV8301-HC-C2-KIT))
* Константы из кода выделяются **жирным шрифтом** (пример: **MINIMUM\_DELTA\_POSITIONS\_TO\_COUNT\_ANGLE\_SPEED**)
* Переменные и термины из формул выделяются **жирным шрифтом** (пример: **const &Delta;T**)
* Имена программных модулей даются **жирным шрифтом** (пример: **модуль задания траектории**)
* Имена файлов выделяются ***курсивным жирным шрифтом*** (пример: ***../utils/graph\_view/Project.exe***)
* Имена функций из кода выделяются *курсивом* (пример: *motorNew::countAngleSpeed()*)

Участки исходного кода выделяются отдельно:

    #include <cstdlib>
    
    void function1()
    {
        exit(1);
    }
